# ChessCoach — Project Rules

## UI Rules
- **NEVER use left-side accent borders** (e.g., `border-left: 3px solid var(--accent)`). Use subtle background tints or rounded containers instead. No vertical accent strips on cards/notes/boxes.
- Keep the UI clean, modern, and minimal. Avoid anything that looks "AI-generated" or overly decorated.
- LLM responses should always be short and crisp (2-4 sentences max for chat, 1-2 for hints).
- Never show numerical evaluations to the user — always use natural language.
